# Eurobarometer Longitudinal Analysis — Extended (4 waves, 2012–2024)

**Author:** Camilla Bonomo  
**Extends:** Gnambs & Appel (2019) — *The impact of social robots on public attitudes toward robotics in Europe*

---

## Overview

This project replicates and extends Gnambs & Appel's (2019) longitudinal analysis of European attitudes toward robots and artificial intelligence, using four Eurobarometer survey waves (2012, 2014, 2017, 2024). The extension adds:

- **Wave 4** (EB 101.4 / SP EB 554, April–May 2024)
- **Hofstede's Uncertainty Avoidance Index (UAI)** as a country-level predictor
- **Italy focus**: longitudinal trajectory, demographic subgroup comparisons, and country-specific random effects
- A **two-item comparable composite** (`rob2item = rob1 + rob2`) enabling direct cross-wave comparison across all four waves

The United Kingdom is excluded from all analyses for cross-wave consistency (absent from wave 4 post-Brexit).

## Analysis Pipeline

Set the working directory to `project/` before running any script:

```r
setwd("path/to/project")
```

Scripts must be run in order. Each script sources `0_Start.R` and loads `dat.Rdata` independently, so they can also be rendered individually.

| Step | Script | Description | Runtime |
|------|--------|-------------|---------|
| 0 | `0_Start.R` | Helper functions (auto-sourced) | — |
| 1 | `1_Load_data_extended.R` | Load raw data, recode variables, run MICE (m=20) | ~30–60 min |
| 2a | `2_1_Descriptives_extended.R` | Descriptive stats, omega reliability | ~5 min |
| 2b | `2_2_Measurement_invariance_extended.R` | Configural → scalar CFA across waves | ~10 min |
| 3 | `3_Current_and_changes_extended.R` | Weighted means, t-tests, attitude change | ~5 min |
| 4 | `4_Predictors_extended.R` | Multilevel models (lmer), pooled across m=20 | ~15 min |
| 5 | `5_Plots_extended.R` | 15 publication-ready figures | ~20–30 min |
| 6 | `6_Document_results_extended.R` | Render all scripts to HTML | cumulative |

To render all scripts to HTML at once, run script 6. **If `data/dat.Rdata` already exists, MICE is skipped automatically** in script 1 (the most time-consuming step).

## Key Variables

| Variable | Description | Range | Waves |
|----------|-------------|-------|-------|
| `rob1` | General appraisal of robots/AI | 0–3 | 1–4 |
| `rob2` | Robots/AI help people | 0–3 | 1–4 |
| `rob3` | Robots doing hard/boring jobs | 0–3 | 1–4 (wording changed in wave 4) |
| `rob` | Three-item composite (rob1+rob2+rob3) | 0–9 | 1–3 only |
| `rob2item` | Two-item comparable composite (rob1+rob2) | 0–6 | 1–4 |
| `feel1–feel4` | Specific application domains | 0–9 | 1–3 only |
| `UAI` | Hofstede Uncertainty Avoidance Index | country-level | time-invariant |
| `LAT`, `LONG` | Country geographic coordinates | — | time-invariant |

Higher values = more positive attitudes throughout.

## Notes

- **MICE**: Multiple imputation uses m=20 datasets, CART method, with `rob` and `rob2item` as passive composites. Seed is fixed (42) for reproducibility.
- **Wave 4 limitations**: `feel1`–`feel4` items are not available on a comparable 1–10 scale in wave 4; `rob3` wording changed in 2024 ("boring/repetitive jobs" vs. "hard/dangerous jobs"), making it non-comparable. All analyses including wave 4 use `rob2item` exclusively.
- **Croatia (HR)**: Absent from wave 1 (2012) and excluded for that wave only.
- **Cyprus (CY)**: UAI value (65) sourced from Hofstede Insights online tool, absent from standard datasets.
